## Fitur
- Dashboard dengan greeting otomatis sesuai waktu
- Clock realtime
- Memory Match dengan timer, level kesulitan, dan penyimpanan skor ke `localStorage`
- Tic Tac Toe mode dua pemain dan AI

## Tech Stack
- React
- Vite
- Tailwind CSS v4 via `@tailwindcss/vite`
- React Router DOM
- localStorage
